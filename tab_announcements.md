## DockSec is now an OWASP Lab Project!

We are thrilled to announce that DockSec has been officially accepted as an OWASP Lab Project. This milestone marks a significant step in our mission to make container security accessible and actionable for developers worldwide.

- **New Documentation**: Updated guides for CI/CD integration.

Stay tuned for more updates as we continue to evolve DockSec!
